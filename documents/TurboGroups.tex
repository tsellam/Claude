\begin{abstract}
In this paper, we introduce \textit{Claude}, a generic query generation system.
With Claude, the users leads, and the system
explores. Here is how it works. The users assign a value to each tuple in the
database.  This value can come from the raw data, a calculation, or manual
effort. Claude's job is to crunch the data and present the ``big picture'': it
explains how the target is distributed across the database, and what influences
it.
\end{abstract}
